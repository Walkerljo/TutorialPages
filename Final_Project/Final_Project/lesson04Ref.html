Web design basics

Content in the middle will vary from embeded videos, questions, quizzes, interactive lessons, etc.
Page will need next, back, home, and topics buttons to help them navigate through the tutorial,
or exit to the main topics menu for the tutorial site. There will also be a breadcrumb trail
that users can use to navigate back to previous information they need to solve problems. There will be 2 versions a version with a white header and footer, as well as white buttons,
and a version with black header and footer and buttons, depending on the topic.
This is a site that is designed with a mobile device such as an ipad in mind, for use
by students in a classroom to interact with tutorial material
